# BobExperiments
Experimentos com o Bob
Neste site você encontrará artefatos utilizados para experimentar o Bob, um agentic AI da IBM.
Os artigos estão publicados no Linkedin, e a cola entre os trechos de códigos no GIT e o artigo você encontra no meu site pessoal em 
www.glaucoreis.com.br


### Arecibo 
Neste exemplo foi utilizado o programa microinvader, um jogo criado como exemplo de programa baseado em microserviços loosed coupled, mas sem documentação., e o Bob gerou artefatos UML e uma explicação detalhada do programa, em apenas 10 minutos. Aprenda a gerar uma documentação completa a partir dos códigos apenas

### JurassicBob
Como utilizar apenas uma gota de sangue, ou um snippet de uma classe para gerar um código completo ? Vamos utilizar o Bob como em Jurassic Park, para gerar um código completo com informações esparsas.
